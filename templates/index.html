/login

描述

登录教务系统并写入cookie值方便其他api调用

参数

是否必须 参数名 参数说明
必须 username 登录教务系统的用户名
必须 password 登录教务系统的密码

返回值说明

字段 类型 说明
status string 状态
message string 状态说明
cookies dict cookie值

调用示例

http://api.chaiziyi.com.cn/jwgl/login?username=123&password=123

JSON示例

登录成功 

{
  "message": "操作成功!",
  "status": "200",
  "cookies": {
    "JSESSIONID": "9574F3932E840F44738C1A568F67xxxx.kingo154"
  }
}

登录失败 

{
  "status": "402",
  "message": "帐号或密码有误!"
}

/scores

描述

查询教务系统上的个人成绩，提供按学期查找、按学年查找和查找全部三种方式

参数

是否必须 参数名 参数说明
非必须 sjxz 时间限制(1为查询全部，2为按学年查找，3为按学期查找)，默认为1
非必须 xn 学年(sjxz为1时忽略此项)，默认为2016，表示2016-2017学年
非必须 xq 学期(sjxz为1和2时忽略此项，0为第一学期，1为第二学期)，默认为0

返回值说明

字段 类型 说明
status string 状态
message string 状态说明
yx string 院(系)/部
xzbj string 行政班级
xh string 学号
xm string 姓名
dysj string 打印时间
allgrade list 所有成绩(包含各学期名和学期成绩)

调用示例

http://api.chaiziyi.com.cn/jwgl/scores?sjxz=3&xn=2015&xq=1

/schedule

描述

按学期查找查询教务系统上的课程表信息

参数

是否必须 参数名 参数说明
非必须 xn 学年，默认为2016，表示2016-2017学年
非必须 xq 学期(0为第一学期，1为第二学期)，默认为0

返回值说明

字段 类型 说明
status string 状态
message string 状态说明
kcb list 课程表
kcm string 课程名
sksj string 上课时间(上课周数和小课节数)
skdd string 上课地点(可能为null)
rq string 日期(格式为kxx，第一个数字表示星期几，第二个数字表示第几节大课，每天分为5节大课，上午两节下午两节晚上一节，如k21就表示星期二的第一节大课)
ls string 老师名字

调用示例

http://api.chaiziyi.com.cn/jwgl/schedule?xn=2015&xq=1

JSON示例

查询成功 

{
  "status": "200",
  "message": "我的课表",
  "kcb": [
    {
      "kcm": "xxx",
      "sksj": "1-20[1-2]",
      "skdd": "xxx",
      "rq": "k21",
      "ls": "xxx"
    },
    {
      "kcm": "xxx",
      "sksj": "1-20[3-4]",
      "skdd": null,
      "rq": "k22",
      "ls": "xxx"
    }
  ]
}

没有检索到记录 

{
  "message": "没有检索到记录!",
  "status": "401"
}

登录信息过期 

{
